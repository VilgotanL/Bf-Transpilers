VBF 2 Specification VBF 2 on Esolangs

by VilgotanL

Interpreter/Transpiler

The tape is identical to bf's tape, 8-bit wrapping.

Comments start with #
Example:

."Hello" #prints hello

Instructions:

Name Description Example(s)
+n Adds n (default 1) to the current cell +, +14
-n Subtracts n (default 1) from the current cell -, -14
>n Moves the pointer right n times (default 1) >, >14
<n Moves the pointer left n times (default 1) <, <14
=n Sets the current cell to n (n required) =0, =14
, Input a character as an ascii value into the current cell (same as bf) ,
. Print the character with the ascii value of the current cell (same as bf) .
.n Print the character with the ascii value of n .65
."string" Print a C-style string, can contain backslash escapes (\n, etc) ."Hello, world!\n"
: Print the current cell's value as a number in base 10 :
n[ Begin while not n (default 0) [code], 0[code], 5[code]
] End while not n ]
n{ Begin while n (default 0) {code}, 0{code}, 5{code}
} End while n }
n( Begin if n (default 0) (code), 0(code), 5(code)
n!( Begin if not n (default 0) !(code), 0!(code), 5!(code)
) End if )

n is optional, n defaults to 1 for most instructions, and 0 for control-flow

n can be a number (42) or a character ('H')
if it is a character, the character's ascii value is used.
Example:

='A' #sets the current cell to 65
