VBI Docs

VBI is a brainfuck extension including better control-flow (while, if) and insert/remove inbetween cells.
NOTE: VBI has different control-flow than VBF, for example, !() and () are switched.

Extra commands:

Instruction Description
![code] While zero
(code) If not zero
!(code) If zero
} Insert a cell to the right of this cell, initialized with a value of 0
{ Remove a cell to the right of this cell

Patterns

  • NOT: }!(>+<)>[-<+>]<{
  • something: insert VBI code here 2
